  • signal 函数原型
#include <signal.h>
void (*signal(int signo, void (*func)(int))) (int);

该函数需要两个参数，返回一个函数指针。
返回的函数指针指向的函数无返回值(void)，有一个整型参数。
第一个参数signo为整数，标示何种信号;
第二个参数void (*func)(int)为函数指针，它指向的函数需要一个整型参数 ，无返回值。

作为参数的函数指针指向的函数指向信号处理程序，作为返回值的函数指针指向之前的信号处理函数。

用typedef简化如下：

#include <signal.h>
typedef void (*sighandler_t)(int);
sighandler_t signal(int signum, sighandler_t handler);

查看本机的.h文件：

//in /usr/include/signal.h
typedef void (*__sighandler_t) (int);

//in /usr/include/bits/signum.h
#define SIG_ERR ((__sighandler_t) -1)       /* Error return.  */
#define SIG_DFL ((__sighandler_t) 0)        /* Default action.  */
#define SIG_IGN ((__sighandler_t) 1)        /* Ignore signal.  */

signal函数有趣的一点是：如果不尝试改变信号处理方式则不能得到目前的信号处理方式
